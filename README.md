# Thesis Defense Expert

学术答辩辅助技能，为 Claude Code 提供专业的答辩准备能力。

## 功能特点

| 图标 | 特点 | 说明 |
|------|------|------|
| 🌍 | 任意技术栈支持 | 自动识别 Java Spring Boot、Python Flask/Django、Node.js、Go 等项目 |
| 🎓 | 差异化服务 | 根据本科/硕士/博士答辩类型自动调整内容深度 |
| 📝 | 完整答辩材料 | 生成答辩稿 + PPT 大纲 + 时间分配建议 |
| ❓ | Q&A 准备 | 高频问题库 + 回答策略指导 |
| 💬 | 答辩技巧 | 开场白/过渡语/应对策略/结束语模板 |

## 安装方式

### 方式一：复制 SKILL.md 文件（推荐）

1. 下载本仓库中的 `SKILL.md` 文件
2. 将其放入 Claude Code 的 skills 目录：
   - **macOS/Linux**: `~/.claude/skills/`
   - **Windows**: `C:\Users\你的用户名\.claude\skills\`
3. 重启 Claude Code 即可使用

### 方式二：通过 Claude Code 命令安装

在 Claude Code 中使用 Skill 工具加载：

```
/skill thesis-defense-expert
```

### 方式三：社区共享（待 Claude Code 社区功能上线）

未来可通过 Claude Code 技能市场一键安装。

## 使用方式

在 Claude Code 中输入：

```
/thesis-defense-expert
```

或描述你的需求，如：

> "帮我准备答辩"

## 工作流程

```
┌─────────────────────────────────────────────────────────┐
│  1. 确认信息                                            │
│     答辩类型、输出格式、分析范围                          │
├─────────────────────────────────────────────────────────┤
│  2. 提取架构                                            │
│     扫描源码结构、数据模型、核心业务逻辑                    │
├─────────────────────────────────────────────────────────┤
│  3. 生成大纲                                            │
│     基于论文和代码生成标准答辩结构                          │
├─────────────────────────────────────────────────────────┤
│  4. 内容填充                                            │
│     生成答辩稿、PPT建议、代码逻辑描述                       │
├─────────────────────────────────────────────────────────┤
│  5. 输出结果                                            │
│     生成完整的 大纲和建议.md 文件                          │
└─────────────────────────────────────────────────────────┘
```

## 输出内容

- 📜 答辩稿（口头陈述完整文字稿）
- 🎯 PPT 每页建议（标题 + 核心要点 + 备注）
- ⏱️ 时间分配指导（根据答辩类型定制）
- ❓ 高频 Q&A 问题库 + 回答策略
- 💡 答辩技巧指导（开场白、过渡语、应对策略、结束语）
- 💻 核心代码逻辑描述

## 适用场景

- 🎓 毕业论文答辩准备
- 📚 课程设计答辩准备
- 🏛️ 学术会议报告准备
